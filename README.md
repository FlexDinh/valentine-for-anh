# Valentine's Day App for Ánh ❤️

Một món quà nhỏ dành tặng cho người đặc biệt nhất.

## Tính năng
- **Màn hình khóa**: Nhập mật mã ngày kỷ niệm để mở khóa.
- **Đếm ngược**: Tính thời gian bên nhau từ ngày 10/10/2025.
- **Hiệu ứng**: Tim bay, pháo hoa, và hộp quà bí mật.
- **Thư tình**: Lời chúc ngọt ngào dịp Valentine.

## Cách sử dụng
Mở file `index.html` trên trình duyệt (khuyên dùng điện thoại) để trải nghiệm tốt nhất.
